Why isn't this one unified taxonomy?
NCSL, AAF, Brennan Center, and FPF each use their own, non-aligned status and category schemas — see Sources & Methods. Merging them into a single made-up taxonomy would misrepresent what each source actually says, so every row here shows its source's own text verbatim, just tagged so you can filter it.
